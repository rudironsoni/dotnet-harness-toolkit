# deep-wiki

Comprehensive wiki generation toolkit that provides:

- slash commands for wiki generation and research workflows
- specialized agents for architecture, writing, and deep research
- supporting skills for changelog, onboarding, llms.txt, AGENTS.md, and deployment

## Use this skill when

- you need to generate a full project wiki
- you want architecture docs with Mermaid diagrams and source citations
- you need onboarding docs for multiple audiences
- you want llms.txt and AGENTS.md generation

## Primary commands

- `/deep-wiki:generate`
- `/deep-wiki:crisp`
- `/deep-wiki:catalogue`
- `/deep-wiki:page <topic>`
- `/deep-wiki:research <topic>`
- `/deep-wiki:onboard`
- `/deep-wiki:llms`
- `/deep-wiki:agents`
- `/deep-wiki:build`
- `/deep-wiki:deploy`
- `/deep-wiki:ado`

See `README.md` in this folder for full plugin details.
